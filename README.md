## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Browser
- Editor (VS code)
- Install Jest testing Library by running the command below

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git https://github.com/frank1738/Front-End-Test-V2.git
```

### Installation (Install depedencies)

```sh
  npm install
```

### Usage

- Open the Html file on your browser

### Run tests

To check linters, run the following command:

```sh
  npm npx stylelint "**/*.{css,scss}"
```
